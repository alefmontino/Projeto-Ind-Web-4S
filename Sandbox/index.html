Sobre a Disciplina

Neste semestre, a disciplina de Programação WEB irá nos guiar pelo desenvolvimento de aplicações modernas para a web. Serão abordados conceitos essenciais e boas práticas de programação voltadas para o ambiente web, usando uma combinação de tecnologias que facilitarão o desenvolvimento, a responsividade e a interação com o usuário.

Durante o curso, estudaremos as seguintes tecnologias e ferramentas:

  • HTML e CSS: A base para estruturar e estilizar páginas web.
  • JavaScript: Linguagem de programação essencial para tornar as páginas interativas.
  • Bootstrap: Biblioteca de CSS que facilita o design responsivo e moderno das páginas.
  • React: Biblioteca JavaScript para construir interfaces de usuário dinâmicas e reativas.
  • APIs REST: Integração de dados com fontes externas usando padrões de comunicação web.
  • Git e GitHub: Ferramentas para controle de versão e colaboração em projetos.

Este conteúdo fornecerá as habilidades e o conhecimento necessários para desenvolver aplicações web profissionais e nos preparará para os desafios do mercado de trabalho na área de tecnologia.
